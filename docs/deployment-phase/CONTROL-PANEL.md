# Control Panel Plan

This document tracks the future control panel for Desktop Dev Cat.

It is intentionally a later-stage feature and should be implemented only after the basic app behavior is finished.

## Purpose

Give the user a simple place to manage the app after installation without needing to edit files or use hidden gestures.

## Suggested Controls

- show or hide the cat
- pause or resume behavior
- reset cat position
- toggle always-on-top
- startup on Windows
- open logs
- open version and build info

## Suggested Sections

- General
- Behavior
- Startup
- Diagnostics

## Notes

- keep the control panel lightweight
- avoid mixing deployment settings with cat animation logic
- keep the panel easy to reach from the tray or a settings button
- make sure the feature remains optional and does not block the core pet experience
