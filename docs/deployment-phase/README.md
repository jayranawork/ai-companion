# Deployment Phase

This folder contains the documents for the post-MVP deployment work.

The deployment phase will begin only after the basic project functionality is complete.

## Goal

Turn Desktop Dev Cat into a distributable Windows app that another user can install and run cleanly.

## What belongs here

- installer and `.exe` packaging notes
- control panel or settings window planning
- tray and startup behavior
- update and versioning strategy
- release testing checklist

## What does not belong here yet

- core cat behavior work
- basic animation and drag polish
- initial renderer and window foundation work

Those items stay in the main implementation phases until the app is stable enough to package.

## Expected output

- a usable Windows installer
- a basic in-app control panel for user settings
- release notes and packaging instructions
- smoke-test checklist for final builds
